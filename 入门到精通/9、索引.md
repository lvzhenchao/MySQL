# 索引：加快查询速度

# 分类：
- 普通索引：key【index】(字段)
- 唯一索引：unique key()
- 主键索引：primary key()
- 全文索引：fulltext() 注意引擎myism:支持英文
- 组合索引：index(多个字段)
- 外键约束：foreign key(字段) REFERENCES 表名(字段名)； innodb 支持【拥有外键的表的数据，和数据类型都需要一样】依附关系

# desc 表名：查看表结构
# show index from 表名：查看索引
