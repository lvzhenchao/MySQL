# 数据类型

## 数值类型
- 整数类型：按数值大小来分

`
    注意有无符号   一个字节：8位，2的8次方
    tinyint  一个字节 2的8次方
    smallint 二个字节 2的16次方
    mediumint 三个字节 2的24次方
    int 四个字节  2的32次方
    bigint 八个字节 2的40次方
`
- 小数类型
`
    float  单精度（浮点，不精确） 8位精度  4字节  m总个数，d小数位
    double 双精度（浮点，不精确） 16位精度 8字节  m总个数，d小数位

    decimal 定点，精确    m总个数，d小数位 （4,2） 12.41

`
